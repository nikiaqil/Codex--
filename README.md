# Yunze 赛事预测记录系统

这是一个本地/Cloudflare 静态网页，用于记录世界杯竞彩预测、赛果、复盘和模型修正。

## 当前内容

- 完赛 / 预测中两个标签页
- 完赛记录按时间从晚到早排序
- 预测中记录按时间从早到晚排序
- 复盘模型修正库默认收缩，可点击展开
- 支持胜平负、让球胜平负、比分、总进球、半全场预测
- 支持体彩开售玩法筛选
- 支持二串一到六串一推荐
- 已复盘 067-072 小组末轮比赛
- 使用最新模型预测 073-076 淘汰赛首批比赛

## 部署

仓库根目录的 `index.html` 即完整网站文件，可直接用于 GitHub Pages 或 Cloudflare Pages。

预测和复盘内容仅用于记录与模型复盘，不构成投注建议。
